<h1 align="center">Date/Time API</h1>

<p align="justify">
    Welcome to the <b>Date/Time API</b> repository! This repository provides hands-on demos explaining the limitations
    of legacy <b>Date</b> and <b>Calendar</b> classes and how the modern <b>java.time</b> package offers improved
    solutions. It covers various <b>Date</b> and <b>Time</b> classes, formatting techniques, and duration calculations
    to handle date-time operations effectively.
</p>
